# Earth OL System UI

一个模拟人生游戏化的启动日志系统，具有完整的 HUD、日志管理和时间线功能。

## 功能特性

- ✨ HUD 状态栏：HP、精神值、金币、等级、Buff/Debuff
- 📅 玩家生日系统：显示"出生日期"、年龄等级、运行时间
- 📝 日志管理：支持自动生成和手动添加
- 🎯 多种日志类型：system、checkpoint、combat、economy、social、status、achievement
- ⌨️ 打字机效果：最新日志逐字显示
- 📊 时间线展示：按年-月-日组织，支持折叠
- 🎨 赛博朋克风格 UI

## Frame 计算说明

Frame 数 = (当前时间 - 出生日期) 的总秒数 × 60 FPS

例如：出生 25 年 = 788,400,000 秒 × 60 = 47,304,000,000 帧
